xyzzy を導入する

editor がないとどうにもならない。 時間を見て emacs をいれるとして、とりあえず xyzzy をいれておく

  • 現状でうまく動いている xyzzy のディレクリを新しい環境のどこかにほうりこむ
    （ここでは ~/bin/ にいれることにした）
  • xyzzy ディレクトリに path を通しておく。たとえば、起動時に実行するバッチファイルに次のように書いておけばいい 
             set HOME=d:/users/i.ootsk2
         set path=%PATH%;%HOME%/bin;%HOME%/bin/busybox;%PATH%/bin/xyzzy
  • ついでに、環境変数 XYZZYHOME を設定しておく 
             set XYZZYHOME=%HOME%/bin/xyzzy

こまかなところで不具合が起きるかもしれないが、それはおいおい対応する。 とりあえずは、これで最低限の動きはするはず
