Python を導入する

管理者権限のない windows マシンに Python を導入できないか、とおもってさがしたら、 Embedded Python というものがあることを知った。

ここやここを参考にすれば、 できそうだ。

Python のインストーラを Python の公式ページ からダウンロードします。Windowsembeddablepackage(64bit)というやつですね。

おとしてきた zip ファイルを解凍してできあがったディレクトリ（以下では、便宜上 embed ディレクトリ といいます）を適当なところ、たとえば ~/bin/にほうりこむ。

つぎに、embed ディレクトリにある python311._pth というファイルの #import site という行の # を削除します。

それから、embedディレクトリ自体とその中にある Scripts というディレクトリに PATH を通します。

もう少しやることがあります。 

$ curl -sSL https://bootstrap.pypa.io/get-pip.py -o get-pip.py

これで embed ディレクトリに get-pip.py がおちてきます。続いて、やはり embed ディレクトリで次のコマンドを打ちます。

./python.exe get-pip.py

これで Scripts ディレクトリに pip.exe ができあがります。

pip をアップグレードします。

$ pip install --upgrade pip setuptools

いろいろなパッケージは、つぎのようにして、おいおいインストールすればいいでしょう。

$ pip install pandas

これで pandas というライブラリがインストールされます。
